# mssh

MSsh — реализация некоторых возможностей SSH посредством отправки и получения электронных писем.

- [ ] JMAP echo с корректным формированием цепочки
- [ ] Слой безопасности с использованием GPG
- [ ] Мультиплексирование
- [ ] Другие способы взаимодействия (передача файлов, команды)
- [ ] IMAP + SMTP транспорт
- [ ] Клиент
- [ ] Транспорт через передачу файлов
- [ ] Интеграция с SSH-ключами и параметрами
- [ ] Переброс портов

Описание архитектуры находится в файле ARCHITECTURE.md
